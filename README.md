# Mini-vue3

本仓库为复现vue3框架的源码和文档，供有兴趣阅读源码的人一起学习，具体的文档请查看[mini-vue3文档](https://tingshine.github.io/mini-vue3/)

## 目录说明

- packages  源码仓库
  - reactive    响应式
    - _tests_  测试
    - basehandlers
    - reactive
    - dep.ts
    - effect.ts
    - ref.ts
    - index.ts
  - shared    工具库
- docs 文档目录
## 教程

```bash
# 安装依赖
npm install

# 测试
npm run test

# 文档开发
npm run docs:dev

# 文档打包
npm run docs:build
```
